# World Life Expectancy Analysis (SQL Project 1)

**Project Overview**  
Cleaned and analyzed global life expectancy data (2007–2022). Removed duplicates, fixed missing values, and explored key trends using SQL.

**Tools & Skills Used**
- MySQL
- Data Cleaning (ROW_NUMBER, UPDATE JOIN)
- Window Functions
- Aggregations & Grouping
- Exploratory Data Analysis (EDA)

**Key Insights**
- Highest 15-year life expectancy gains: Haiti (+28.7), Zimbabwe (+23.0)
- Developed countries average 79.2 years vs 67.1 years in Developing countries
- High GDP countries have ~9.5 years higher life expectancy than low GDP countries
